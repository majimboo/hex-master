Windows-First Hex Editor

Hex Master

A keyboard-friendly desktop hex editor with structural editing, typed inspection, byte-pattern search, and a Rust core behind a Qt shell.

Download Latest Releases Repository

Current Feature Set

  • Hex and text-side editing with insert and overwrite modes
  • Find next, find all, typed-value search, and structured replace
  • Inspector dock with grouped numeric and time interpretations
  • Bookmarks, hashes, session restore, and recent files

Release Model

  • Semantic versioning from a single CMake version source
  • GitHub Actions CI for Windows build and Rust tests
  • Tagged releases publish downloadable Windows archives
  • GitHub Pages provides a lightweight public landing page
